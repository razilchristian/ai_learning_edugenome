# 🧬 EduGenome

### Unlock Your Learning DNA

EduGenome is an AI-powered personalized learning platform designed to adapt education according to each student’s unique learning behavior, pace, and performance.

---

## 🚀 Live Demo

👉 https://ailearningedugenome.vercel.app/

---

## 📌 Overview

Traditional education follows a **one-size-fits-all approach**, which does not work for every student.

EduGenome solves this by introducing a **data-driven, AI-assisted learning system** that personalizes content and improves learning outcomes.

---

## 🧠 Core Concept – Learning DNA

EduGenome builds a **Learning DNA Profile** for each student based on:

* 📊 Quiz performance
* ⏱ Learning speed
* 🧠 Mistake patterns
* 📈 Engagement time
* 📚 Assignment quality

👉 This helps generate **personalized learning paths**

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/edugenome.git

# Navigate to project
cd edugenome

# Install dependencies
npm install

# Run server
node server.js
```

---

## 🔐 Environment Variables

Create a `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_api_key
```
